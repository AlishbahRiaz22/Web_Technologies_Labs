This repository contains my lab work and practice exercises for the **Web Technologies** course. The goal of these labs is to develop a strong foundation in front-end web development, including webpage structure, styling, and basic interactivity.

---

## 🚀 Technologies & Tools

- **HTML5** – Semantic markup and page structure  
- **CSS3** – Styling, layouts, responsive design  
- **JavaScript** – DOM manipulation, event handling, basic interactivity  
- **VS Code** – Code editor  
- **Browser Developer Tools** – Debugging & testing

---

## 📝 Topics Covered

- HTML structure: headings, lists, tables, forms  
- CSS fundamentals: selectors, box model, flexbox, grid  
- Responsive design basics  
- CSS animations and transitions  
- JavaScript fundamentals: variables, functions, loops  
- DOM manipulation & event handling  
- Simple interactive features (buttons, form validation)

---


Each folder contains the HTML, CSS, and JS files for that particular lab or project.

---

## 🎯 Purpose

This repository demonstrates my hands-on learning in front-end web development. It serves as a portfolio of practical exercises and small projects, highlighting:

- Understanding of web page structure and design  
- Ability to implement responsive and interactive features  
- Organized approach to coding and documentation
